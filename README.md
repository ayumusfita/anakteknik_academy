# anakteknik_academy
Simple API and Web Automation using Ruby and cucumber
​
## Setup
​
### Dependencies
- RVM
- Ruby 2.5.1
- Git
- Bundler
​
### Install RVM & Ruby
1. RVM
​
  ```
  curl -L get.rvm.io | bash -s stable
  source ~/.bash_profile
  ```
​
  Then run `rvm requirements` and follow the instructions
​
2. Ruby 2.5.1
​
  ```
  rvm install 2.5.1
  ```
​
## Setup Gem
​
  ```
    rvm use 2.5.1@anakteknik_academy --create
    gem install bundler
    bundle install
  ```
